## Shared prompt contract

- Square raster app/program icon candidate
- Image 1 is an identity reference, not an edit target
- Original mascot combining a small agile wild beast with a formidable fantasy warrior
- Preserve large pointed ears, cyan/teal eyes, broad toothy lower face, orange and blue-gray contrast
- One centered subject, bold silhouette, generous safe margin, readable at 32px
- No text, watermark, UI mockup, scenery, franchise logo, official emblem, trademark, or exact existing character copy

각 candidate는 표의 direction만 바꾼 별도 prompt로 생성했다.
